# 界面设计与交互规范文档 (UI/UX Specification)

本文档详述了 CKCP LAMP 应用程序的核心界面结构、组件定义及布局规范，旨在为 UI 设计师提供准确的功能参考。

---

## 1. 视觉风格总则 (Visual Guidelines)

-   **核心风格**: 2.0 时代的毛玻璃主义 (Glassmorphism 2.0)。
-   **背景**: 使用动态弥散背景 (Diffuse Background)，具有深紫色、蓝色或青色的渐变感。
-   **容器**: 所有的主要区块必须包裹在 `GlassCard` 中，具备毛玻璃半透明、细微边框和内部阴影。
-   **色彩**:
    -   主色 (Primary): `#2196F3` 或具有活力的蓝色。
    -   危险色 (Danger): `#F44336`。
    -   文字颜色: 在深色模式下使用纯白 (`#FFFFFF`) 或浅灰 (`#B0B0B0`)。
-   **交互**: 所有的按钮、卡片、滑块在通过点击/拖动时需伴随平滑的动画过渡。

---

## 2. 氛围灯控制页面 (Ambient Light Page)

该页面采用 **左右双栏结构**，适配平板或桌面端的横向展示。

### 2.1 左侧边栏 (320px)
-   **设备连接卡片 (`Device Connection`)**:
    -   显示当前连接状态 (已连接/未连接/搜索中)。
    -   显示设备名称、硬件版本、软件版本、车型。
    -   提供“断开连接”操作。
-   **灯光基础设置卡片 (`Light Settings`)**:
    -   **分区亮度开关**: 切换“总亮度”控制与“精细分区”控制。
    -   **亮度控制区**:
        -   *总亮度模式*: 一个 0-10 级的水平滑块。
        -   *分区亮度模式*: 分为“驾驶位”、“副驾位”、“后排”三个带图标的独立滑块。
    -   **开关控制区**: “开”、“关”、“车灯跟随”三个等宽的 Segmented Button。

### 2.2 右侧主内容区 (Expanded)
-   **模式切换器**: 位于顶部，包含“单色”、“多色”、“律动”三个选项卡的切换容器。
-   **单色面板 (Mode 0)**:
    -   **颜色选择器 (HSL Picker)**: 大面积的环状或方形调色盘，带有一个中心十字光标。
    -   **颜色预览与应用**: 显示当前颜色预览块、十六进制色码 (`#RRGGBB`) 和“应用”按钮。
    -   **预设颜色网格**: 常用色彩 (红、绿、蓝、黄、白等) 的圆形快捷按钮。
-   **多色面板 (Mode 1)**:
    -   **动态开关**: 全局动态模式开关。
    -   **主题网格**: 卡片式布局，每个卡片包含一张主题示意图（如“海洋”、“森林”）和功能名称。
-   **律动面板 (Mode 2)**:
    -   **效果网格**: 卡片式布局，重点展示音频同步的动效示意图。

## 3. 工厂模式页面 (Factory Page)

该页面采用 **单列居中布局** (Max Width: 800px)，侧重于功能性的表单设置。

### 3.1 核心板块
-   **设备注册 (`Device Registration`)**:
    -   VIN 码：17 位大写字母/数字输入框 + “注册”按钮。
    -   车型编号：0-255 数字输入框 + “设置”按钮。
    -   功能编号：0-255 数字输入框 + “设置”按钮。

-   **LED 硬件配置 (`LED Config`)**:
    -   **双列对称布局**: 左侧展示“驾驶侧”，右侧展示“副驾侧”。
    -   **精细区域**:
        -   驾驶侧：主驾、左前门、左后门。
        -   副驾侧：副驾、右前门、右后门。
    -   **单项控制**: 每个区域包含“灯珠数量 (Slider/Input)”和“反向开头 (Switch)”。

-   **音频设置 (`Sound Settings`)**:
    -   **音源二选一**: “内置麦克风”与“原车音响”的大型 Radio Card。
    -   **灵敏度分级**: 1-5 级的阶梯式按钮。

-   **高级联动功能 (`Advanced Features`)**:
    -   垂直列表，包含：迎宾模式、车门联动、车速响应、转向联动、空调联动、仪表预警。
    -   每个项目包含：图标、功能名称、以及一个右侧的 Toggle Switch。

-   **调试日志 (`Remote Control`)**:
    -   一个半透明的黑色背景终端窗口，显示实时收发的十六进制数据流。
    -   底部带有一个手动发送自定义指令的输入框。

-   **危险操作区 (`Danger Zone`)**:
    -   显著的警告色 (红色) 边框，包含“恢复出厂设置”按钮。点击后需弹出二次确认模态框。

## 4. 响应式与状态建议 (UX Recommendations)

1.  **加载状态**: 在设备查询配置或应用升级包时，UI 应展示半透明的加载蒙层和线性/圆形进度条。
2.  **断连反馈**: 蓝牙断开时，所有控制组件应进入半透明禁用状态 (`Opacity: 0.5`)，并显示“设备已离线”的浮窗提示。
3.  **微动效**: 
    -   切换 Tab 时，背景颜色做轻微的偏移。
    -   拖动亮度滑块时，滑块轨道应有发光晕染效果。
